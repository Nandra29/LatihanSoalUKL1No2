# LatihanSoalUKL1No2
Fungsi: Program ini bertujuan untuk mengecek dan menentukan apakah sebuah angka yang dimasukkan user termasuk bilangan ganjil atau genap.
Cara Kerja:-Program meminta user memasukkan angka.
           -Angka tersebut dibagi 2, dan program hanya melihat sisa pembagiannya (modulus).
           -Jika sisanya adalah 0,angka tersebut adalah genap.
           -Jika sisanya adalah 1,angka tersebut adalah ganjil.
